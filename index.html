Lecture 1 Theory

Open file Dialogue

  • Used to get a file location
  • Takes the user to the windows file explorer when the relevant code is executed.

How to Show/display it

  • OfdName.ShowDialog(), where ofdName is the name you have to your open file dialog control

How to get the location from the open file dialog

  • use ofdName.FileName, this will return a string
  • e.g string path = ofdName.FileName; this will store the location of the file e.g "C\users\Desktop\audio.mp3"

Windows Media Player

  • Ussed to play MP3 (music)/MP4 (videos)

What to do if Windows Medial player doesn't appear on your toolbox

  • Open the toolbox
  • Right click anywhere on the toolbox window
  • Select Choose Items
  • Select COM Compoents
  • Scroll down until you find the Windows Media Player and Tick it
